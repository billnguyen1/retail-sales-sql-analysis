# retail-sales-sql-analysis

Retail Sales Data Analysis

Project Overview:

This project analyzes retail sales data using SQL to identify revenue trends, customer behavior, and top-performing product categories.

Tools Used: SQL (MySQL)

Key Objectives:

- Clean and transform raw sales data
- Analyze monthly revenue trends
- Identify top-selling products and regions
- Calculate customer lifetime value metrics

Key Insights:

- Found highest revenue months and seasonal trends
- Identified top 10% customers contributing majority of revenue
- Highlighted underperforming product categories

SQL Techniques Used: JOIN, GROUP BY, Aggregate Functions, Date Functions
